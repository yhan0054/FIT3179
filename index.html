Internet Users around the world in 2016 Report

Author: Yalun Han

Date: 16/10/2022

This data visualisation indicates the number of internet users in the world in 2016. Through this visualisation, you would also see the country with the most amount of people using network and the country with the least amount of the people using network in 2016.

Note: There are some countries that are not included in the dataset, which is why some of the places is grey.

From the map above, it indicates the number of internet users around the world in 2016, it has been splited into few colours, light blue, blue, darker blue and etc. Due to the large number of people in the map, a domain that contains larger values would suit the map. From the map, it shows the country with the most internet users in 2016 is China with 746,662,194 people using network, the country with least of people using network is Kiribati with only 15,672 people.

Note: The countries are in descending order, this is for better showing the overall trend, which clearly indicates which country has the most internet users.

From the bar chart, it shows the first 10 countries with the most number of internet users in 2016, apparently the top country is China, this is because of its population base. The thing that is surprising is that Japan, as the country that does not have a very large landscape, still managed to have more than one hundred million people that have internet access. However, big countries like United Kindom could not make into the top 10 countries, which is weird, but based on the population, it is understandable for why it did not manage to get in.

From this line chart, it indicates the top 10 countries with the most internet users in 2020, this is to have a comparison with the bar chart above. However, China and India are still the top 2 countries with the most people using network in the country.
